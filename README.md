# Antigen-Guided Antibody Design Analysis

This repository contains a static GitHub Pages project summarizing a computational antibody design run from `Zolbetuximab_tarlatamab_test`.

## Contents

- `index.html` - the single-page project website
- `assets/figures/` - readable report figures and split subfigures
- `assets/report/` - English Word report
- `data/` - CSV and FASTA result tables
- `structures/top_candidates/` - PDB files for the top candidates

## Key result

The run produced 80 final candidates. The top-ranked candidate is `samples_design_4_dldesign_3_best` with a composite affinity score of 15,360.5.

## Important note

This is a computational analysis summary only. It is not experimental validation, clinical evidence, or a safety assessment.
